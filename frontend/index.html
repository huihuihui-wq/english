    Shadowing Tips:
    1. Loops — Set how many times each sentence repeats
    2. Delay — Add a pause before replaying the same sentence (echoic shadowing)
    3. Gap — Pause before moving to the next sentence
    4. Start with 0.75x speed and 1-2 loops, then increase speed
    5. Use Delay > 0 for echoic memory training (shadowing after a short pause)
